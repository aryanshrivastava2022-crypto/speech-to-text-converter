# Speech-to-Text Converter
A Python application that converts spoken words into text using speech recognition.

# Features
- Convert speech into text
- Uses microphone input
- Simple command-line interface
- Automatic speech recognition
- Basic error handling

# Requirements
- Python 3.x
- SpeechRecognition
- PyAudio

# Installation
Install the required libraries:
pip install SpeechRecognition
pip install PyAudio
> **Note:** On some operating systems, installing PyAudio may require additional setup.

# How to Run
1. Download the project files.
2. Open a terminal in the project folder.
3. Run:
speech_to_text.py
4. Speak clearly into your microphone.
5. The recognized text will be displayed on the screen.
